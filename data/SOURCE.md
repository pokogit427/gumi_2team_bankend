# 데이터 원본 정보

이 문서는 현재 워크스페이스에 포함된 JSON 파일만을 기준으로 확인한 데이터 출처 및 사용 기준입니다.

---

## 1. 데이터명

- 구미/경북권 관광정보 JSON 집합
- 파일명 기준으로 확인된 콘텐츠 유형: 관광지, 레포츠, 문화시설, 쇼핑, 숙박, 여행코스, 음식점, 축제공연행사

## 2. 확인 가능한 메타 정보

| 항목 | 값 | 비고 |
|---|---|---|
| 제공기관 | 확인 필요 | JSON 파일 내부에 직접적인 기관명 메타는 확인되지 않음 |
| 원본 출처 | 확인 필요 | 파일 내부에 원본 API URL 또는 출처 메타는 확인되지 않음 |
| 라이선스 | 확인 필요 | 파일 내부에 라이선스 문구는 확인되지 않음 |
| 공공누리 유형 | 확인 필요 | 워크스페이스의 기존 문서에서는 제3유형으로 언급되지만, 현재 JSON 파일 자체는 직접 확인되지 않음 |
| 수집일 또는 제공일 | 확인 필요 | JSON 항목의 `createdtime`/`modifiedtime`은 있으나, 데이터셋 전체의 수집일은 파일 내부에 명시되지 않음 |
| 사용 범위 | 프로젝트 내 지역정보 조회, 검색, 챗봇 컨텍스트 구성에 사용 | 원본 데이터는 수정하지 않음 |
| 원본 데이터 변경 금지 여부 | 확인 필요 | 현재 파일은 프로젝트에서 원본 그대로 사용하며, 백엔드에서는 메모리 캐시/응답 변환만 수행할 예정 |

## 4. 데이터 집합 요약

- 권역명: 구미_경북권
- 총 데이터 건수: 1,667건
- 각 파일은 `items` 배열을 포함하고 있으며, 이 배열을 백엔드에서 로딩할 예정입니다.
- 현재 확인된 데이터는 공공데이터 형태의 관광정보 JSON이며, 커뮤니티 게시글 데이터는 포함되어 있지 않습니다.
